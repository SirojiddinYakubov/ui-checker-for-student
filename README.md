# UI Checker

## Installation

### Clone repository

```
git clone https://github.com/SirojiddinYakubov/ui-checker-for-student.git
```

### Up project

```
docker compose up -d --build
```

### Open browser

```
http://0.0.0.0:8000
```

## Additional

> **_NOTE:_**  Install a pre-commit hook to write and check your code before committing.

```
pre-commit install
```
